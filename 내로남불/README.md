# 🤔 내로남불 테스트

> 나는 얼마나 이중잣대를 가지고 있을까? 재치있는 질문들로 알아보는 내로남불 성향 테스트

## 📖 프로젝트 소개

**내로남불 테스트**는 "내가 하면 로맨스, 남이 하면 불륜"의 줄임말인 '내로남불' 성향을 재미있게 측정할 수 있는 웹 기반 심리 테스트입니다. 

일상생활에서 자주 마주하는 상황들을 통해 자신과 타인에게 적용하는 기준이 얼마나 다른지 확인할 수 있습니다.

## ✨ 주요 기능

### 🎯 현재 구현된 기능

- **📝 12가지 재치있는 질문**: 일상생활 기반의 현실적인 상황 제시
- **⚖️ 이중 상황 비교**: 같은 행동을 '내가 할 때'와 '남이 할 때'로 나누어 측정
- **📊 6가지 결과 타입**: 성인군자형부터 내로남불 레전드형까지 다양한 결과
- **📈 상세 분석**: 이중잣대 성향, 자기합리화 정도, 타인 관대성 수치화
- **📱 반응형 디자인**: 모바일, 태블릿, 데스크톱 최적화
- **🎨 아름다운 UI/UX**: 그라데이션과 애니메이션 효과
- **⌨️ 키보드 지원**: 1-4 숫자키로 빠른 답변 선택 가능
- **📋 결과 공유**: 클립보드 복사 및 네이티브 공유 API 지원
- **🌙 다크모드 지원**: 시스템 설정에 따른 자동 테마 변경
- **♿ 접근성 최적화**: 키보드 네비게이션 및 모션 감소 옵션

### 📍 현재 진입점 (Entry Points)

- **메인 페이지**: `index.html` - 테스트 소개 및 시작
- **테스트 진행**: 12개 질문 순차 진행 (각 질문당 2개 상황, 4개 선택지)
- **결과 화면**: 개인화된 내로남불 성향 분석 결과

## 🏗️ 프로젝트 구조

```
내로남불-테스트/
├── index.html              # 메인 HTML 파일
├── css/
│   └── style.css          # 스타일시트 (반응형, 다크모드 지원)
├── js/
│   ├── questions.js       # 질문 데이터 및 결과 분석 로직
│   └── main.js           # 메인 JavaScript 로직
└── README.md             # 프로젝트 문서
```

## 🎮 사용법

1. **테스트 시작**: 메인 화면에서 "테스트 시작하기" 버튼 클릭
2. **질문 답변**: 각 질문마다 두 가지 상황(내가 할 때 vs 남이 할 때)에 대해 4점 척도로 답변
3. **결과 확인**: 7개 질문 완료 후 개인화된 내로남불 성향 결과 확인
4. **결과 공유**: 공유 버튼으로 결과를 다른 사람과 공유

### ⌨️ 키보드 단축키

- **1-4 숫자키**: 첫 번째 상황 답변 선택
- **Shift + 1-4**: 두 번째 상황 답변 선택  
- **Enter**: 다음 질문으로 진행
- **Escape**: 메인 화면으로 돌아가기

## 📊 결과 분석 시스템

### 결과 타입 (6가지)

1. **😇 성인군자형** (0-20%): 거의 내로남불하지 않음
2. **🤔 균형잡힌 현실주의자형** (21-40%): 적당한 수준의 내로남불
3. **😅 선택적 관대형** (41-60%): 상황에 따라 다른 잣대 적용
4. **🙄 이중잣대 마스터형** (61-80%): 제법 강한 내로남불 성향
5. **🤪 내로남불 레전드형** (81-100%): 매우 강한 내로남불 성향
6. **😱 역내로남불형** (음수): 자신에게 더 엄격한 특이 케이스

### 상세 분석 지표

- **이중잣대 성향**: 자신과 타인에 대한 기준 차이
- **자기합리화 정도**: 자신의 행동을 정당화하는 경향
- **타인 관대성**: 타인에 대한 포용도 및 이해도
